### Common Tasks

**Task:** Add a new page

1. Create `client/src/pages/NewPage.tsx`
2. Add route in `client/src/App.tsx`
3. Add to navigation (AppSidebar or AdminSidebar)
4. Add test IDs for Playwright
5. Follow MT Ocean theme guidelines

**Files to edit:**
- `client/src/pages/NewPage.tsx`
- `client/src/App.tsx`
- `client/src/components/layout/AppSidebar.tsx`

---

**Task:** Add a new API endpoint

1. Create route handler in `server/routes/feature-routes.ts`
2. Add middleware (auth, RBAC if needed)
3. Validate input with Zod schemas
4. Use service layer for business logic
5. Add to API documentation

**Files to edit:**
- `server/routes/feature-routes.ts`
- `server/services/FeatureService.ts` (if needed)
- `shared/schema.ts` (if new data models)

---

**Task:** Add a new database table

1. Define table in `shared/schema.ts`
2. Create insert/select schemas with Zod
3. Run `npm run db:push` to apply changes
4. Update storage interface if needed
5. Create API routes for CRUD operations

**Files to edit:**
- `shared/schema.ts`
- `server/storage.ts` (if using storage abstraction)
- `server/routes/*.ts` (API routes)

---

**Task:** Integrate a new AI feature

1. Check if Bifrost supports the provider
2. Add to `bifrost-config/bifrost.yaml` if needed
3. Create service in `server/services/`
4. Use Bifrost base URL for requests
5. Add API endpoints
6. Create frontend components

**Files to edit:**
- `bifrost-config/bifrost.yaml`
- `server/services/NewAIService.ts`
- `server/routes/ai-routes.ts`
- `client/src/components/ai/NewAIComponent.tsx`
