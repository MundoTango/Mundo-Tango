### 1.3 Dark Mode Implementation

**Critical Requirement:** 80% of users prefer dark mode (research data)

```typescript
// client/src/components/ThemeProvider.tsx

import { createContext, useContext, useEffect, useState } from 'react';

type Theme = 'light' | 'dark';

const ThemeContext = createContext<{
  theme: Theme;
  toggleTheme: () => void;
}>({ theme: 'dark', toggleTheme: () => {} });

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  const [theme, setTheme] = useState<Theme>(() => {
    // Default to dark mode (80% user preference)
    const saved = localStorage.getItem('theme');
    return (saved as Theme) || 'dark';
  });

  useEffect(() => {
    document.documentElement.classList.toggle('dark', theme === 'dark');
    localStorage.setItem('theme', theme);
  }, [theme]);

  const toggleTheme = () => setTheme(t => t === 'light' ? 'dark' : 'light');

  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
}

export const useTheme = () => useContext(ThemeContext);
```

### 2.1 Video Hero Section

**Research:** "Video-First Design dominates 2024-2025"

**Implementation:**
```typescript
// client/src/components/VideoHero.tsx

import { useState, useRef, useEffect } from 'react';
import { Play, Pause } from 'lucide-react';
import { Button } from '@/components/ui/button';

interface VideoHeroProps {
  videoUrl: string;
  posterUrl?: string;
  overlayGradient?: string;
  children: React.ReactNode;
}

export function VideoHero({ 
  videoUrl, 
  posterUrl, 
  overlayGradient = 'from-black/60 via-black/30 to-black/60',
  children 
}: VideoHeroProps) {
  const [isPlaying, setIsPlaying] = useState(true);
  const videoRef = useRef<HTMLVideoElement>(null);

  useEffect(() => {
    // Auto-play on mount (muted)
    videoRef.current?.play();
  }, []);

  const togglePlay = () => {
    if (videoRef.current) {
      if (isPlaying) {
        videoRef.current.pause();
      } else {
        videoRef.current.play();
      }
      setIsPlaying(!isPlaying);
    }
  };

  return (
    <section className="relative min-h-screen flex items-center justify-center overflow-hidden">
      {/* Video Background */}
      <video
        ref={videoRef}
        className="absolute inset-0 w-full h-full object-cover"
        poster={posterUrl}
        loop
        muted
        playsInline
      >
        <source src={videoUrl} type="video/mp4" />
        <source src={videoUrl.replace('.mp4', '.webm')} type="video/webm" />
      </video>

      {/* Gradient Overlay */}
      <div className={`absolute inset-0 bg-gradient-to-b ${overlayGradient}`} />

      {/* Content */}
      <div className="relative z-10">
        {children}
      </div>

      {/* Play/Pause Control */}
      <button
        onClick={togglePlay}
        className="absolute bottom-8 right-8 z-20 p-3 rounded-full bg-white/20 backdrop-blur-md hover:bg-white/30 transition-all"
        aria-label={isPlaying ? 'Pause video' : 'Play video'}
      >
        {isPlaying ? <Pause className="h-6 w-6 text-white" /> : <Play className="h-6 w-6 text-white" />}
      </button>
    </section>
  );
}
```

```typescript
// client/src/components/ScrollReveal.tsx

import { motion } from 'framer-motion';
import { useInView } from 'framer-motion';
import { useRef } from 'react';

interface ScrollRevealProps {
  children: React.ReactNode;
  delay?: number;
  direction?: 'up' | 'down' | 'left' | 'right';
}

export function ScrollReveal({ 
  children, 
  delay = 0, 
  direction = 'up' 
}: ScrollRevealProps) {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true, margin: "-100px" });

  const variants = {
    hidden: {
      opacity: 0,
      y: direction === 'up' ? 40 : direction === 'down' ? -40 : 0,
      x: direction === 'left' ? 40 : direction === 'right' ? -40 : 0,
    },
    visible: {
      opacity: 1,
      y: 0,
      x: 0,
      transition: {
        duration: 0.6,
        delay,
        ease: [0.25, 0.1, 0.25, 1],
      },
    },
  };

  return (
    <motion.div
      ref={ref}
      initial="hidden"
      animate={isInView ? "visible" : "hidden"}
      variants={variants}
    >
      {children}
    </motion.div>
  );
}
```

**SECTION 2: THE BIG FOUR ORCHESTRAS**

Four Golden Age orchestras (1935-1955) with distinct personalities:

**1. Juan D'Arienzo - "El Rey del Comp√°s" (King of the Beat)**
- **Sound:** Sharp, staccato, driving rhythm with crystal-clear pulse
- **Tempo:** Fast (63-68 BPM), brought dancers back to milongas in 1935
- **Characteristics:** Pulsing percussive piano, strong marcato beats
- **Dancing:** High energy, quick weight changes, perfect for crowded floors and beginners

**2. Carlos Di Sarli - "El Se√±or del Tango" (The Gentleman)**
- **Sound:** Elegant, smooth, majestic with powerful walking beat
- **Characteristics:** Lush piano-led orchestrations, bandoneons = rhythm, violins = melody
- **Era:** 1950s recordings especially beloved
- **Dancing:** Smooth flowing walk, long steps, slow weight transfer ("unroll the foot")

**3. An√≠bal Troilo - "Pichuco" (Champion of the Singer)**
- **Sound:** Bandoneon virtuoso, complex, playful, emotionally rich
- **Characteristics:** Sharp contrasts between staccato and lyrical moments
- **Innovation:** Elevated singers to full lyrical sections (not just refrains)
- **Dancing:** Playful creative interpretation, most complex of Big Four

**4. Osvaldo Pugliese - "San Pugliese" (Tango at Its Zenith)**
- **Sound:** Passion, drama, intensity‚Äîoff the scale
- **Characteristics:** Simultaneous driving beat AND soaring melody
- **Famous:** "La Yumba" beat (heavy emphasis on 1 and 3)
- **Dancing:** Big dramatic movements, open embrace, giros and suspensions

**SECTION 4: THREE RHYTHMS**

**Tango (2/4 or 4/4 time)**
- Dramatic, melancholic, romantic
- Slow deliberate beat with pauses
- Long powerful steps, full weight transfer
- Walking emphasizes elegance

**Vals (3/4 time - waltz)**
- Lilting, flowing, dreamlike, joyful
- Continuous circular movements (giros)
- Soft knees, "dancing on cotton"
- Swirling continuous motion

**Milonga (2/4 syncopated)**
- Fast, energetic, playful, cheeky
- Small quick sharp steps in tight embrace
- African rhythmic influences
- Choppy, linear (not circular)

---

**SECTION 5: DANCE STYLES**

**Tango de Salon**
- Elegant social style, closed embrace that opens for ochos/giros
- Straight axis, smooth flow, respects line of dance
- Best for: social dancing, competitions

**Milonguero Style**
- Compact for crowded floors, very close "apilado" embrace (chest-to-chest)
- Shared axis with slight lean, small refined steps
- Best for: crowded milongas

**Tango Nuevo**
- Emphasizes movement exploration, long steps, varied spatial relationships
- Open elastic embrace, includes sacadas, ganchos, boleos, volcadas
- Famous dancers: Gustavo Naveira, Chicho Frumboli, Pablo Ver√≥n

**Stage Tango (Fantasia)**
- Choreographed for performance, theatrical elements with lifts
- Popularized by Juan Carlos Copes (1950s)
- NOT for social dancing

---

**SECTION 6: LOS C√ìDIGOS (MILONGA ETIQUETTE)**

**El Cabeceo** - The eye contact invitation system
- Make eye contact (mirada), subtle head nod
- Eliminates awkward verbal rejections
- Used at start of tanda (not during cortina)

**La Tanda** - Set of 3-4 songs (same orchestra, same era)
- Commit to dancing full tanda
- Typical: 4 tangos, 3 valses, or 3 milongas
- Only leave mid-tanda for emergency

**La Cortina** - The "curtain" (30-60 sec non-tango music)
- Separates tandas, clears the floor
- Never dance during cortina
- Escort partner back to seat

**La Ronda** - Counter-clockwise "lane" around floor
- Stay in your lane, don't cut across
- Outer = faster, inner = slower
- Leaders face traffic, protect followers
